# CFNADE-keras

* Keras implementation of CF-NADE.

## How to run

* Use `python3.5`.
```
git clone git@github.com:JoonyoungYi/CFNADE-keras.git
cd CFNADE-keras
virtualenv .venv -p python3
. .venv/bin/activate
pip install -r requirements.txt
python data_prep.py
python run.py
```
